Cyrillic To Latin is a plugin for transforming Serbian Cyrillic .po files into Latin .po files.

== Description ==

Cyrillic to latin only transform _rs_RS files, don't use this plugin if you want to use both translation, for this you have different types of plugins.
Use this plugin only with Serbian language,  because i't won't work with others.
Transform any cyrillic translation of wordpress language file. It can transform themes, admin's and woocommerce translation files.
